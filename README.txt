------------------------Inventory Scripts--------------------------------------

GetDBDataNames.py - Pulls the names of the files in the specified database.


FindDuplicates.py - Finds the data duplicated between frameworks in the DB.


ListMapData.py - Lists information for all .mxd files in the specified folder.


testsForIncreCount.py - Tests for a recursive function used in comparison.py


comparison.py - Does the comparison between mxd data and db data, displaying the 
	important relationships between the two.



------------------------------Steps--------------------------------------------

Step 1: Change folders and files in the src/Constants.py file to your computer

Step 2: Change where output will go from GetDBDataNames.py file if needed. Run GetDBDataNames.py in the ArcMap Terminal. 

Step 3: Run Main.py
